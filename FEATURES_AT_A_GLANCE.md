### API Endpoints
```
PUBLIC:
  POST   /ask                    → Ask question
  POST   /interview              → Single Q&A
  POST   /interview/start        → Start session
  POST   /interview/answer       → Submit answer
  GET    /student/:id/analytics  → Performance
  GET    /problems/random        → Random problem
  POST   /code/execute           → Run code

TRAINER ONLY:
  GET    /trainer/leaderboard    → Rankings
  GET    /dashboard/overview     → Metrics
  GET    /dashboard/weak-students → Help needed
```

## 💡 USAGE EXAMPLES

### Start Interview
```javascript
// 1. Select subject
subject = "JavaScript"

// 2. Start session
POST /interview/start
{ subject: "JavaScript", studentId: "123" }

// 3. Get first question
→ "What is a closure?"

// 4. Answer
POST /interview/answer
{ sessionId: "xyz", answer: "A closure is..." }

// 5. Get feedback + next question
→ Feedback: "Great! Score: 8/10"
→ Next: "Explain event bubbling"
```

### Solve Problem
```javascript
// 1. Get random problem
GET /problems/random?difficulty=easy

// 2. Write code
code = "function twoSum(nums, target) { ... }"

// 3. Execute
POST /code/execute
{ language: "javascript", code, testCases }

// 4. Get results
→ Passed: 3/3 tests
→ Score: 100%
```
