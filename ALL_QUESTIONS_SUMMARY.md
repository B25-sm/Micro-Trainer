## 🎨 QUESTION DIFFICULTY LEVELS

### Easy Questions (Beginner Level)
- **Purpose:** Test basic understanding
- **Format:** "What is X?" or "What does Y do?"
- **Answer Time:** 10-15 seconds
- **Examples:**
  - "What is a closure?"
  - "What is JSX?"
  - "What is Node.js?"

### Medium Questions (Intermediate Level)
- **Purpose:** Test practical knowledge
- **Format:** "How do you X?" or "What is the difference between X and Y?"
- **Answer Time:** 20-30 seconds
- **Examples:**
  - "How do you implement JWT authentication?"
  - "What is the difference between useMemo and useCallback?"
  - "How does the Event Loop work?"

### Hard Questions (Advanced Level)
- **Purpose:** Test deep understanding and real-world application
- **Format:** "Explain X with an example" or "What are the implications of Y?"
- **Answer Time:** 30-45 seconds
- **Examples:**
  - "Explain the React reconciliation algorithm."
  - "What are the phases of the Node.js Event Loop?"
  - "How do you implement complex form validation?"

## 🔄 ADAPTIVE DIFFICULTY SYSTEM

### How It Works:
1. **Initial Assessment:** Starts with easy questions
2. **Performance Tracking:** Monitors recent 3 answers
3. **Dynamic Adjustment:**
   - Score < 4/10 → Easy questions
   - Score 4-7/10 → Medium questions
   - Score ≥ 7/10 → Hard questions

### Benefits:
- ✅ Personalized learning experience
- ✅ Prevents frustration (too hard) or boredom (too easy)
- ✅ Optimizes learning curve
- ✅ Builds confidence progressively
